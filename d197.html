► Version Control - D197


Hey all, todays write up is for D197 or better known as Version Control


I am not going to go into the requirements for this PA because they are clearly listed in the course.

Take everything you learn from this class and commit (haha) it to memory because this is not the last time you will use git in your life if this is your first.


  1. The first thing I would recommend is using Learn Git Branching.This course will walk you through everything you need to know about Git especially for this course.
  2. If you make a mistake inside the Working branch, instead of scraping it you can salvage it. Here is a funny resource to show you how to rollback changes.

  3. Here is a video course from Programming with Mosh. It's only 1 hour and will give you another visual guide.

  4. When you are ready to begin your PA make sure you follow the GitLab How-To to set up your project correctly.

  5. Once your branch is set up follow each step of the PA and do exactly as it says. If you forget a command refer to the github docs or ask the magic 8 ball.


I recognize this writeup included many more learning resources instead of actual examples. That being if I posted said examples it would essentially be giving you the answer and I was recently informed that doing this could cause trouble for me as a current WGU student. But seriously if you forget a command just ask the magic 8 ball.


This is what worked for me, you may need more or less. Don't be afraid to deviate from this.


Good luck!
